Reduce wrist movement & type faster on any keyboard

I started to feel slight discomfort in my wrists while typing to I replaced my regular

my regular keyboard with a kinesis advantage.

There was a problem though, when I was working remotely it was

annoying to carry the laptop and the keyboard with me.

So I started this project so that I can

type any key on a classic keyboard without moving my wrists.

TURN THIS - INTO THIS

  • Highly customizable
